# LEGAL SHIELD AI

AI Powered Cyber Crime Detection & Legal Assistant

## Problem
Cybercrime such as phishing, fraud, and online blackmail is increasing. Many victims do not know how to identify threats or report cybercrime legally.

## Solution
LEGAL SHIELD AI is an AI powered platform that helps users detect cyber threats and understand applicable cyber laws.

## Features
- Scam Detection using AI
- AI Legal Advisor
- Evidence Analysis
- Automatic Complaint Generator

## Technology Stack
Frontend: HTML CSS JavaScript  
Backend: Python Flask  
AI: Generative AI + NLP  
Database: MongoDB / Firebase

## System Workflow
1. User uploads suspicious link or screenshot
2. AI analyzes threat
3. System identifies applicable cyber law
4. Complaint draft is generated

## Future Scope
- Integration with Government Cyber Crime Portal
- Mobile App
- Real time scam detection
